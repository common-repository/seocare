== Description ==

SeoCare helps the admin to create SEO-Friendly posts. It has several rules that can be edited in the options. After creating the post SeoCare calculates a grade for the post.
That way you can easely see on what points you better can change your article so it would score better on SEO. 

== Installation ==

This section describes how to install SeoCare and get it working.

1. Unzip SeoCare.zip file
2. Upload the SeoCare folder (The whole folder) to your wp-content/plugins folder.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. (Optional) Change the plugin configuration in the SeoCare option menu
